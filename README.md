# Gianni's Portfolio

This repository contains a personal portfolio website for Gianni Petrizzo. The site showcases projects, a resume, and contact information, presented with a stained-glass-inspired visual theme and clean, accessible HTML/CSS.

## What this site is
- A multi-page static portfolio site (Home, About, Portfolio, Resume).
- Focuses on showcasing technical projects, research work, and professional background.
- Designed to be responsive and accessible with an elegant stained-glass aesthetic.

## Features
- Project gallery with category filters.
- Hero sections with visual emphasis and call-to-action buttons.
- Resume page with downloadable/printable view.
- Contact section with email and phone links.
- Lightweight, dependency-free static HTML + CSS (no build step required).

## Technologies
- HTML5
- CSS3 (custom properties, responsive layout, accessibility enhancements)
- Vanilla JavaScript for small interactive pieces (mobile nav toggle, project filters)
